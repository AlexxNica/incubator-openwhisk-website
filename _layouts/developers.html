Scaling Per-Request & Optimal Utilization

Run your action ten thousand times in a fraction of a second, or once a week. Action instances scale to meet demand exactly, then disappear.

Flexible Programming

Code in different languages like JavaScript/NodeJS, Swift, Python, Java and to run custom logic by executing Docker containers.

Invoke your code synchronously, asynchornously or even periodically.

Use higher-level programming constructrs like sequencing which allows you to declaratively (i.e. without touching your actions code), chain together multiple actions parameter binding which relieves you from the need to hardcode things like credential in your code

And also, Debug your code in realtime.
